## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/LegionC/cable-inventory.git
   ```

2. Open `index.html` in your browser, or serve with any static file server:
   ```bash
   npx serve .
   ```

3. Start managing your cable inventory!

## Usage

- **Inventory Tab**: Add, edit, and manage cable resources
